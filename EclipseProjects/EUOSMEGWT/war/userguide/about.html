EUOSME (version 1.00)

EUOSME is a new metadata editor to create, load and save metadata files following INSPIRE Metadata Implementing Rules (Technical Guidelines based on EN ISO 19115 and EN ISO 19119).

Please send any suggestions for improvements, and feedback on this client to:

inspire-geoportal@jrc.ec.europa.eu
